```text
sql/002_image_archive.sql
```

新增表：

```text
image_archive
```

该表按 DICOM Series 保存脱敏影像归档记录，原始 DICOM 文件仍由 Orthanc 存储。

## 升级步骤

已有 `v0.1.0-docker` 数据库时，先应用迁移：

```powershell
Get-Content sql/002_image_archive.sql | docker compose exec -T postgres psql -U rt_research -d rt_research
```

然后重建并启动：

```powershell
docker compose up -d --build api web
```

重新跑 DICOM ETL 后，既有 CT/CBCT Series 会写入 `image_archive`：

```powershell
docker compose --profile etl run --rm etl python -m etl.run_etl
```
