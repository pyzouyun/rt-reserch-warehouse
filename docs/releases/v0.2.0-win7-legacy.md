# v0.2.0-win7-legacy 版本说明

本版本线目标是同一个离线安装包支持：

- Windows 7 SP1 64 位
- Windows 10 64 位

本版本不依赖 Docker Desktop。

## 计划运行时

- Python 3.8.10 x64
- PostgreSQL 10.23 x64 binaries
- Orthanc 1.11.3 x64 Windows runtime
- NSSM 2.24
- Inno Setup 6.3.x 安装器

## 服务

- `RTResearch-PostgreSQL`
- `RTResearch-Orthanc`
- `RTResearch-API`

legacy 版由 API 服务同时提供 Web UI 和 `/api/v1`。

## 端口

- Web UI：`http://localhost:8080`
- API：`http://localhost:8080/api/v1`
- API docs：`http://localhost:8080/docs`
- Orthanc Web：`http://localhost:8042`
- Orthanc DICOM：`4242/tcp`
- PostgreSQL：`127.0.0.1:5432`

## 用户手册

Win7/Win10 无 Docker 离线版安装、启动、DICOM 接收、ETL、MOSAIQ CSV 导入、备份和排错说明见：
