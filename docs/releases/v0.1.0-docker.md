# v0.1.0-docker 固定版本说明

`v0.1.0-docker` 是 Docker 版原型的首个固定交付版本。

## 范围

该版本包含：

- Orthanc DICOM Server。
- PostgreSQL 研究数据库。
- FastAPI 后端。
- React Web 工作台初版。
- Python ETL。
- MOSAIQ CSV 导入。
- Windows Docker 打包脚本。

## 数据边界

- 不写回 Monaco。
- 不写回 MOSAIQ。
- 不写回 XVI。
- 不写回 Elekta 加速器。
- 不保存患者姓名、身份证、电话、住址等直接身份标识。

## 访问地址

- Web UI：`http://localhost:8080`
- API 文档：`http://localhost:8000/docs`
- Orthanc：`http://localhost:8042`
- DICOM C-STORE：`4242`

## 后续版本

`v0.1.1-docker` 在该版本基础上增加了 Research Workbench V2、XVI/CBCT 影像归档和更完整的中文文档。
