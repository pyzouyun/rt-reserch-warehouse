# Research Use Cases

- Automatic plan quality control: compare RTPLAN, RTDOSE, structures, prescriptions, and DVH metrics.
- CBCT pseudo-CT: link planning CT, CBCT, registrations, and delivered fraction context for model development.
- Setup error analysis: study XVI registration shifts, CBCT frequency, immobilization, site, and machine effects.
- Dosimetric prediction: train dose or DVH predictors using structures, plans, and outcomes.
- Radiomics: build curated CT/CBCT/RTSTRUCT cohorts while maintaining pseudonymous joins.
- Toxicity and outcome modeling: combine DVH metrics, treatment fractions, diagnosis, and clinical outcome rows.
- Adaptive radiotherapy research: compare longitudinal CBCT anatomy and delivered fractions against original plans.
