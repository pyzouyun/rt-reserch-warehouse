# v0.2.0 Statistics 发布说明

发布日期：2026-05-25

## 版本目标

本版本把放疗研究数据仓库从“数据接收、解析、归档”推进到“研究队列统计与脱敏导出”。它仍然保持只读临床源系统、不写回 Monaco/MOSAIQ/XVI/加速器的边界。

## 新增能力

- 新增统计 API：
  - `GET /api/v1/statistics/cohort-summary`
  - `GET /api/v1/statistics/prescription-distribution`
  - `GET /api/v1/statistics/imaging-summary`
- 新增脱敏 CSV 导出 API：
  - `GET /api/v1/export/patients-csv`
- Web 工作台新增“统计”页面：
  - 队列患者数、年龄、分次数、CBCT 数、计划 CT 数概览。
  - 性别、队列标签、入组状态、审核状态分布。
  - 处方剂量、治疗技术、治疗部位、治疗设备分布。
  - 患者级计划 CT / CBCT 归档概览。
  - 一键导出脱敏患者 CSV。
- 新增下一版技术设计文档：
  - `docs/statistics_v0.2_technical_design.md`
  - `docs/research_stats_prompt.md`
  - `docs/plans/2026-05-25-statistics-v0.2-implementation.md`

## 数据安全边界

导出的 CSV 只包含 `research_patient_id`、统计分组字段、处方摘要、分次数和影像数量，不包含明文姓名、身份证、电话、住址，也不导出原始 `PatientID`。

统计接口只读取研究库中的脱敏结构化数据，不访问或写回临床生产系统。

## 使用方式

启动服务后访问：

```text
http://localhost:8080
```

打开左侧“统计”页面即可查看统计结果。点击“导出患者 CSV”可下载脱敏研究表。

命令行导出示例：

```powershell
Invoke-WebRequest -Uri "http://localhost:8000/api/v1/export/patients-csv" -OutFile ".\patients_export.csv"
```
